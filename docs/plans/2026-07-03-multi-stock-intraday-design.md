# 多股票与盘中相对价值分析设计

## 范围

分析池扩展为东方财富（300059）、同花顺（300033）、指南针（300803）和财富趋势（688318）。每只股票独立训练四组模型、选择参数和回测，不共享回归系数。日线保存最近十年或上市以来全部历史；指南针和财富趋势上市不足十年，严禁补造上市前数据。

盘中数据采用公开接口能够获得的最长分钟历史，默认5分钟。分钟线仅负责判断实时价格相对最近一个已完成日线模型中枢的位置，并保存原始记录；它不参与十年日线回归，也不宣称具备十年覆盖。若主分钟接口不可用，降级到新浪近期分钟线并记录数据区间。

## 数据与输出

共享市场指数和市场活跃度数据，每只股票分别建立 `data/<code>/` 与 `output/<code>/`。个股目录保存原始日线、分钟线、清洗数据、模型数据、指标、回测、图表和报告。顶层输出生成四股横向汇总、最新信号与通知事件。

通知模块采用可插拔设计。第一阶段生成 `alerts.csv` 和 `alerts.json`，只在强买、买入观察、卖出观察、强卖或风险升级时产生事件，并使用状态文件去重。后续接入飞书、邮件或其他通道时不修改模型逻辑。

## 稳健性

保留扩展窗口样本外预测；三只新增股票分别评估可用样本数。回测结果明确标注交易次数不足、成交额代理和分钟历史长度。分钟线不得回填未来日线信息，盘中理论中枢仅使用上一完成交易日产生的模型参数与当日已知市场信息。
